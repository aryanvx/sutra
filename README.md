# Sutra

An interactive, introspective journey exploring connection, home, and memory through language, place, and personal reflection. Also, my first project written in Swift through Xcode.

## Features

- **Thread of Place**: Tap to reveal Marathi (my mothertongue) words with ripple animations
- **Thread of Language**: Connect Marathi phrases to English translations
- **Thread of Memory**: Create a visual web of personal memories
- **Beautiful Gradients**: Warm color palette (saffron, gold, coral, indigo - stuff that represents my culture)
- **Haptic Feedback**: Tactile responses throughout the journey
- **Text-to-Speech**: Hear Marathi words pronounced

## Built With

- SwiftUI
- AVFoundation (for speech synthesis)
- CoreHaptics (for haptic feedback)

## Requirements

- iOS 17.0+
- Xcode 15.0+

## Installation

1. Clone the repository
2. Open `Sutra.xcodeproj` in Xcode
3. Build and run on simulator or device
